Tutorial Membuat Tabel

Tabel dalam HTML digunakan untuk menampilkan data dalam format baris dan kolom. Struktur dasar tabel terdiri dari elemen <table> yang berfungsi sebagai wadah tabel. Di dalam tabel terdapat dua bagian utama: <thead> untuk mendefinisikan header tabel dan <tbody> untuk konten data.

Bagian <thead> digunakan untuk mendefinisikan header tabel, yang biasanya berisi judul kolom yang menjelaskan jenis data yang akan ditampilkan. Setiap kolom dalam header tabel didefinisikan menggunakan elemen <th>, yang memberikan penekanan visual pada judul tersebut.

Sementara itu, bagian <tbody> berisi konten data dari tabel. Di dalam <tbody>, setiap baris data didefinisikan menggunakan elemen <tr>, dan setiap sel data dalam baris tersebut didefinisikan menggunakan elemen <td>. Ini memudahkan pemisahan antara header dan isi tabel, serta membuatnya lebih mudah untuk dibaca.

Berikut contoh sintax untuk membuat tabel dengan kolom first name, last name, age, dan alamat 

        
        <table>
          <thead>
            <tr>
              <th>First Name</th>
              <th>Last Name</th>
              <th>Age</th>
              <th>Alamat</th>
            </tr>
          </thead>
          <tbody>
            <tr>
              <td>Mia</td>
              <td>Anggreini</td>
              <td>20</td>
              <td>Trihanggo</td>
            </tr>
          </tbody>
        </table>
